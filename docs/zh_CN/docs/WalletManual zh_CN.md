链内转账
--------

链内转账是指在两个CCDEX账户之间，对CCDEX的资产，和在CCDEX上发行的资产，以及CCDEX支持的资产进行转账操作。各类资产在CCDEX账户之间的转账快速、高效。转账过程只发生CCDEX的手续费，综合费率低廉。

跨链资产充值和提取
------------------

跨链数字资产充值，是指通过DCCP跨链协议将来自于其他区块链的数字资产映射到CCDEX的操作，并能够与CCDEX上的其他各类数字资产进行交易和转账。
